# Jogo da Velha (Tic-Tac-Toe) - Java

Este projeto é uma implementação do Jogo da Velha desenvolvida para fins acadêmicos e desafio de projeto da Plataforma de ensino ADA, focada em **Lógica de Programação**, **Orientação a Objetos** e **Tratamento de Exceções** em Java.

## 📝 Funcionalidades
- Tabuleiro 3x3 jogável via console.
- Sistema de turnos entre os jogadores 'X' e 'O'.
- Validação de jogadas (impede marcar casas ocupadas ou fora do limite).
- Detecção automática de vitória ou empate (Velha).
- Tratamento de erros com `try-catch-finally` para garantir que o programa não trave em caso de entradas inválidas.

### ⚙️ Futuras feautures

## 💻 Conceitos Aplicados
- **Encapsulamento:** Separação da lógica de jogo (`JogoDaVelha.java`) da execução do console (`Main.java`).
- **Exceções Personalizadas:** Criação da classe `JogadaInvalidaException` para um controle mais refinado das regras.
- **Estruturas de Dados:** Uso de matrizes bidimensionais para representação do tabuleiro.
- **Boas Práticas:** Código comentado e estruturado conforme os padrões ensinados em aula.

## 🛠️ Como executar
1. Certifique-se de ter o **JDK 21** (ou superior) instalado.
2. Clone o repositório ou baixe os arquivos.
3. Abra o projeto em sua IDE (IntelliJ IDEA recomendada).
4. Execute o arquivo `Main.java` localizado em `src/main/java/org/jogo_da_velha/`.
